üí≥ Credit Cards as Separate Object
While credit cards are stored as Loan records in Symitar (Loan.Type = credit card), they will be synced to a separate Credit Cards custom object in HubSpot for better reporting and segmentation. This enables tracking of credit-card-specific fields like cc_credit_limit, cc_utilization_pct, and cc_last_transaction_date.
‚ö†Ô∏è Important: MANTL Data Limitations
MANTL's API provides minimal application data (applicationId, product, status only). Applicant personal information (name, email, phone, address) is NOT available via MANTL API. Full applicant details are populated from SymXchange after the application is booked to Symitar core.
‚úì Backfill Matching Process
Daily SymXchange sync matches newly opened accounts to existing MANTL Deals by: product type + open date + member name. When matched, the Deal is updated with core_account_number and associated with the Contact.
